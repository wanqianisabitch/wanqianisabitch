```text
TypeError: 'float' object cannot be interpreted as an integer
```

在python2中， '整数 / 整数 = 整数'，以上面的 100 / 2 就会等于 50， 并且是整数。

在python3中， ‘整数/整数 = 浮点数’，也就是100 / 2 = 50.0，使用 '//'就可以达到原python2中'/'的效果。
